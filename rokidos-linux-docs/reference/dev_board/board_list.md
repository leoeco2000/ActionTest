## 参数介绍

* **Amlogic/A113 开发板**
	* **芯片厂商：** *Amlogic*<br>
	* **芯片型号：** *A113*<br>
	* **开发板内部代号：** *banban_m2*<br>
	* **编译框架：** *buildroot*<br>
	**开发板基本参数介绍：**<br>
		* **CPU：4核的A543处理器/1.5Ghz** 
		* **Memory：** 512MB 内存 + 512MB Nand 闪存 
		* **Mic Array：** 4 mic阵列 
		* **Audio：**3.5mm音频输出 
		* **Network：**支持无线网络802.11a/b/g/n/ac
		* **Bluetooth：** 支 持BT 4.1/BLE 
		* **LED：** 彩色 LED*12 
		* **DC in：** USB type-c充电调试接口，5V的电压

<br>
